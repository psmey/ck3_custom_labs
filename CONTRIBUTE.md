## Adding Images

For images exist some constraints for where they are used

| Where          | Contstraint                                  |
| -------------- | -------------------------------------------- |
| Thumbnail      | must be `<mod_dir>/thumbnail.png` and < 1 MB |
| Steam Workshop | < 2 MB                                       |
